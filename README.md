## Descripcion parte 1 HTML: 
Se trabajo de forma colaborativa para el desarrollo de los siguientes puntos:
-Creación de los archivos index.html, proyectos.html, detalle.html, y perfil.html en la rama develop.
-Implementacion de etiquetas semánticas (como <header>, <nav>, <main>, <section> y <footer>.
-Uso de etiquetas de contenido como <h1>, listas (<ul>), tablas (<table>), formularios (<form>).
-Vinculación de documento mediante etiquetas (<a>) para permitir la navegacion fluida. 
-Validacion completa del codigo HTML con W3C.

## Descripcion parte 2 CSS:
Se aplicaron estilos visuales para dar consistencia y funcionalidad a la interfaz:
-Se utilizó una hoja de estilos externa (style.css) ubicada en la carpeta css.
-La estructura general se definió mediante CSS Grid, componentes como el menú de navegación se organizo con Flexbox.
-Se configuraron estados interactivos (:hover)
-Se utilizaron clases e identificadores para aplicar reglas de estilo.
